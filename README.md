This is a sample implementation of restaurant management system with Flask-Restful
Docker image: vikashchy/restaurant-management

#### To build and deploy image
docker-compose up --build -d 

#### Techstack used:
Flask-Restful
Flask-Sqlalchemy
Flask-mail
Docker & Docker Compose
Sqlite
